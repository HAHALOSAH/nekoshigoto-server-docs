Creates a new API key.

createAPIKey(name: string, privilegeLevel: int, flags: NekoShigotoAPIKeyFlags, expiresAt: Date): Promise<NekoShigotoAPIKey>
Parameters
name (string) The name of the API key.
privilegeLevel (int = 0) The privilege level of the API key.
flags (NekoShigotoAPIKeyFlags = new NekoShigotoAPIKeyFlags().decode(0)) The flags of the API key.
expiresAt (Date = null) The time at which the API key expires, as a JavaScript Date. If this is set to null, the API key never expires.
